<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">OpenOlat</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">是一个基于网络的电子学习平台，用于教学、学习、评估和交流，是一个 LMS、一个学习管理系统。 OpenOlat 以其简单直观的操作和丰富的功能集给人留下了深刻的印象。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">复杂的模块化工具包为课程作者提供了广泛的教学可能性。每个 OpenOlat 安装都可以单独扩展、适应组织需求并集成到现有 IT 基础设施中。该架构专为最小的资源消耗、可扩展性和安全性而设计，以保证系统的高可靠性。</font></font></p>
